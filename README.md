# Save Praise (◠‿◠)

a kawaii kaomoji companion that lives in your editor. every time you save, it praises you, and shows your changes.

| diff     | praise |
|----------|--------|
| `+12 -3` | nice save (◠‿◠) |
| `+45`    | you've been busy (°o°) |
| `-8`     | satisfying cleanup (◡‿◡) |
| `~4`     | fine-tuning (⌒‿⌒) |
|          | just checking in? (◕ᴗ◕) |

big additions get excited reactions. deletions get cleanup praise. in-line edits get their own messages. saving with no changes? it notices that too.

the praise appears both in the status bar (with a colored highlight and icon) and as ghost text next to your cursor. it fades after a few seconds.

### what's new in 0.2.0

100+ messages across all categories. context-aware status bar icons and colors. time-sensitive messages for late nights, early mornings, lunch, and friday afternoons. 2% chance of a rare legendary praise. and in-line edits are now properly detected.

### settings

- `savePraise.enabled` - turn it on or off (default: on) 
- `savePraise.duration` - how long the praise stays, in ms (default: 3000)

### toggle

`Ctrl+Shift+P` -> **Save Praise: Toggle On/Off**
